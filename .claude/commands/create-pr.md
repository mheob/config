description: Create a git commit
---

Create a Pull Request:

- the title is written as conventional commit with a max length of 50 characters
- the description is organized in "Summary", "Changes" and "Motivation"
- add the scope as a label (e.g. `eslint :rotating_light:` or `deps :package:` etc.)
  - collect the available list from the github repository
  - multiple label are possible
- prefer `gh` cli over mcp server
